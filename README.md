## Project Start 
Before you run the project, please ensure a `.env` file exists in the root directory. 

Example `.env`
```bash
PORT=3000
MONGO_URI=mongodb://localhost:27017/clothing-store 
JWT_SECRET=yoursecret` 
``` 

Start the server 
```bash 
$ yarn run start:dev
```
